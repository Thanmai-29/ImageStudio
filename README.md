# 🖼️ ImageStudio

ImageStudio is a modern image processing web application built with **React, TypeScript, and Vite**. It provides an easy and efficient way to upload, edit, and download images through a clean and responsive interface.

🌐 **Live Demo:** [https://imagepro-ivory.vercel.app/](https://imagepro-ivory.vercel.app/)

## ✨ Features

- 📤 Upload and preview images
- ✨ Enhance image quality
- 🗜️ Compress images
- ✂️ Crop images
- 🔄 Rotate images
- 🎨 Apply image editing effects
- 🔐 User authentication
- 📱 Responsive design

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- shadcn/ui
- Vitest

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Thanmai-29/ImageStudio.git

# Navigate to the project
cd ImageStudio

# Install dependencies
npm install

# Start the development server
npm run dev
```

## 📂 Project Structure

```text
src/
├── components/
├── pages/
├── hooks/
├── lib/
└── integrations/
```
